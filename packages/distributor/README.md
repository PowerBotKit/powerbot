# @powerbotkit/distributor

> PowerBotKit Distributor package

## Install

Using npm:

```shellscript
npm install @powerbotkit/distributor --save
```

using yarn:

```shellscript
yarn add @powerbotkit/distributor
```

## Cache

`distributor` need a cache. 

Now we provide some cache

1. MemoryCache
2. RedisCache
